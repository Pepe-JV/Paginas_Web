2. Tipografía

Fuente Principal (Títulos)

Georgia / Garamond (Serif)

Serif clásica que aporta elegancia y sofisticación artesanal.

H1 - Flores que hablan por ti

H2 - Nuestras Categorías

H3 - Ramos de flores

Fuente Secundaria (Cuerpo)

Helvetica Neue / Arial (Sans-serif)

Sans-serif limpia y moderna para máxima legibilidad en textos largos.

Párrafo regular: Somos Pétalo&Co, una floristería de barrio con 15 años de experiencia.

Párrafo negrita: Creamos arreglos únicos para bodas y eventos especiales.

Párrafo cursiva: Cada ramo cuenta una historia única.

Escala Tipográfica

  • H1: 3.5rem (56px) - Títulos principales hero
  • H2: 2.5rem (40px) - Títulos de sección
  • H3: 1.8rem (28.8px) - Subtítulos
  • H4: 1.3rem (20.8px) - Encabezados menores
  • Párrafo: 1rem (16px) - Texto base
  • Small: 0.85rem (13.6px) - Textos pequeños

3. Botones

Botón Primario

Fondo verde suave, texto blanco. Hover con color acento.

Botón Secundario

Borde verde, fondo transparente. Hover relleno verde.

Botón Submit

Similar al primario, diseñado para formularios.
